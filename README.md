# AsteroidsReplica
Тестовое задание

УПРАВЛЕНИЕ
===============================
A, D для поворота
W, S для управления ускорением
Пробел для стрельбы

ПРАВИЛА
===============================
Игра ведется до тех пор, пока у игрока остаются жизни
За уничтожение целей начисляются очки
20 - Большой астероид
50 - Средний астероид
100 - Малый астероид
150 - НЛО

ЦЕЛЬ ИГРЫ
===============================
Набрать максимальное количество очков
